# 🎨 AI Art Generator

Generate stunning images from text descriptions using **Stable Diffusion** and **Streamlit**!

## ✨ Features

- 🖼️ **Text-to-Image Generation** - Create unique artwork from detailed descriptions
- ⚙️ **Advanced Controls** - Adjust steps, guidance scale, resolution, and more
- 📊 **Negative Prompts** - Specify what you DON'T want in your image
- 📜 **Generation History** - Access your previous prompts
- ⬇️ **Easy Download** - Save generated images directly
- 🚀 **Lightning Fast** - Get results faster

## 🚀 Quick Start

1. Enter a detailed description of your desired image
2. (Optional) Adjust advanced settings
3. Click "Generate Image"
4. Download your masterpiece!

## 💡 Prompt Tips

### Good Prompts (Detailed & Specific)
```
"A majestic dragon flying over snowy mountains at sunset, 
fantasy art style, highly detailed, cinematic lighting, 
4K quality, by artist name"
```

### Better Prompts (Include everything)
- **Subject**: What do you want?
- **Style**: Art style or artist reference
- **Lighting**: How should it be lit?
- **Quality**: Resolution and detail level
- **Mood**: Emotional tone

## 🛠️ Technologies

- **Stable Diffusion v1.5** - Generative AI model
- **Diffusers** - HuggingFace model pipeline
- **Streamlit** - Web framework
- **PyTorch** - Deep learning framework
- **CUDA** - GPU acceleration

## 📊 Model Info

- **Model**: runwayml/stable-diffusion-v1-5
- **Input**: Text prompts
- **Output**: 512x512 to 768x768 images
- **Speed**: 30-60 seconds per image on GPU
- **VRAM**: 6GB+ (optimized for 8GB)

## ⚠️ Important Notes

- Model downloads (~4GB) on first run
- Requires GPU for reasonable speed
- Generation time depends on steps and resolution
- Results vary based on prompt quality
